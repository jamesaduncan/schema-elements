Microdata API - Test Suite and Examples

Quick Start

The Microdata API provides seamless access to HTML microdata with live DOM synchronization.

// Access microdata from any element const person = document.getElementById('person1').microdata; console.log(person.name); // "John Doe" // Access all microdata on the page const items = document.microdata; console.log(items.length); // Number of microdata items // Fetch microdata from remote URLs const data = await Microdata.fetch('https://example.com/page.html#fragment');

Browser Console Demo

Open your browser console and try these commands:

// This page has some demo microdata document.microdata.length // See how many items // Access first item document.microdata[0] // Access by ID (if items have IDs) document.microdata['demo-item'] // Create new microdata programmatically const div = document.createElement('div'); div.setAttribute('itemscope', ''); div.setAttribute('itemtype', 'https://schema.org/Thing'); div.innerHTML = '<span itemprop="name">Dynamic Item</span>'; document.body.appendChild(div); // It's immediately available div.microdata.name // "Dynamic Item"
