A schema for Schema Properties

This schema defines the structure of a schema property. It includes properties for the name of the property, its type, whether it is required, its cardinality, and a description of the property.

Schema
Property Type Required Cardinality Description
name https://schema.org/Text True One The name of the property in the schema
type https://schema.org/URL True One The type of the property, which can be a URL to a schema or a primitive type.
required https://schema.org/Boolean True One Indicates whether the property is required in the schema.
cardinality https://schema.org/Text True One The cardinality of the property, which can be "One", or "Many".
description https://schema.org/Text False One A description of the property, explaining its purpose and usage in the schema.
