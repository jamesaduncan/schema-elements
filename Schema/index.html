A schema for Schemas

This schema defines the structure of a schema. It includes properties for the name of the schema, its parent schema, and the properties that it defines.

Schema
Property Type Required Cardinality Description
name https://schema.org/Text True One The name of the property in the schema
parent https://schema.org/URL False One The schema that this schema is a child of. This is used to create a hierarchy of schemas.
property https://schema.org/SchemaProperty False Many Each of the properties that the schema defines.
